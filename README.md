# cashz static (Vercel-ready)

This repo contains 3 static pages:
- `/` — import landing
- `/seed` — seed phrase import
- `/open` — open wallet page

## Deploy to Vercel
Import this repo in Vercel. No build command is required (static).

## Local run
Use any static server, e.g.
- `python -m http.server 8080`
then open http://localhost:8080
